# 💰 Gestor de Despesas Pessoais Cloud - 2026

Este é um projeto de **Gestão Financeira** desenvolvido em Python, focado em praticidade, interface moderna e armazenamento seguro na nuvem. 

Este projeto faz parte do meu portfólio de estudante de **Análise e Desenvolvimento de Sistemas (ADS)** na Universidade Anhembi Morumbi.

## 🚀 Tecnologias Utilizadas
* **Python 3.13**: Linguagem principal do projeto.
* **Streamlit**: Para a criação da interface web interativa e responsiva.
* **PostgreSQL (Neon.tech)**: Banco de dados relacional hospedado na nuvem.
* **Dotenv**: Para gestão segura de credenciais e variáveis de ambiente (Segurança).

## 🛠️ Funcionalidades
- Cadastro de receitas e despesas em tempo real.
- Visualização de saldo total atualizado automaticamente.
- Conexão direta com banco de dados remoto (Neon Cloud).
- Filtros de visualização de dados com Pandas.

## 🔒 Boas Práticas e Segurança
O projeto segue as normas de segurança de desenvolvimento, utilizando um arquivo `.gitignore` para impedir que chaves de acesso e senhas de banco de dados sejam expostas publicamente no repositório.
